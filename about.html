About TaxiMeter Pro

A free, privacy-first GPS taxi fare calculator built for passengers everywhere — no app, no account, no data collection required.

Our Mission

TaxiMeter Pro was created with a simple goal: give every passenger the power to verify their taxi fare in real time, using nothing more than a smartphone and a browser.

In many cities around the world, taxi passengers are vulnerable to fare manipulation — broken meters, unofficial routes, or inflated charges. We built TaxiMeter Pro to be a transparent, independent tool that any passenger can open on any device, in any country, and immediately start tracking their ride.

We believe access to fare transparency should be free. That's why TaxiMeter Pro will always be free to use.

Our Story

The Problem

Taxi passengers worldwide often have no way to independently verify whether a meter reading is accurate. In many developing countries, meters are tampered with or simply absent, leaving passengers at the mercy of drivers.

The Idea

We asked: what if any passenger could open a webpage and start a fair, GPS-based meter on their own phone? No app to install, no account to create — just open and go.

The Build

Using the browser's native Geolocation API and OpenStreetMap's open mapping data, we built a lightweight fare calculator that runs entirely client-side. All fare logic, GPS tracking, and route drawing happens in your browser.

Today

TaxiMeter Pro is used daily by passengers across 50+ countries. We continue to improve accuracy, add features, and listen to feedback from our global community of users.

Technology

TaxiMeter Pro is built with pure HTML, CSS, and JavaScript — no frameworks, no heavy libraries, no unnecessary complexity. We use:

Leaflet.js — an open-source JavaScript library for interactive maps, paired with OpenStreetMap tile data.

Browser Geolocation API — the W3C-standard API built into every modern browser for accessing GPS coordinates with user permission.

Haversine Formula — the mathematical formula used to calculate the great-circle distance between two GPS coordinates on Earth's surface.

This combination means TaxiMeter Pro loads fast, runs anywhere, and never needs to send your data to a server.
